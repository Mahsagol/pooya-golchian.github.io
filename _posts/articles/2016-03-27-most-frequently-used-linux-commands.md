***1. tar command examples***

Create new tar archive
{% highlight comments %}
$ tar cvf archive_name.tar dirname/
{% endhighlight %}

Extract from an existing tar archive.
{% highlight comments %}
$ tar xvf archive_name.tar/
{% endhighlight %}
